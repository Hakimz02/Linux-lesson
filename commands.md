# Linux Commands I Use

## Navigation
- ls → list files
- cd → change directory
- pwd → show current location

## Files
- touch file.txt → create file
- cat file.txt → view file
- nano file.txt → edit file

## Permissions
- chmod 644 file → read/write owner, read others
- chmod 755 file → full owner, read+execute others

## Networking
- ip a → show IP address
- ping google.com → test connection

## Services
- systemctl start nginx → start web server
- systemctl status nginx → check service
